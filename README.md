# Random-dad-jokes
Generate a random dad joke with a click of a button

# Created with
  1. React
  2. npm package axios
  3. API call to https://icanhazdadjoke.com/

# How to set up:
Install dependecies:
```bash
cd .\dad-jokes-app\
```
```bash
npm install
```

Start the server:
```bash
npm start
```
